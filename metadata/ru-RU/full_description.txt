Anemo — это приложение для Android, предназначенное для работы с личным локальным хранилищем.
Вместо того чтобы быть отдельным пользовательским интерфейсом файлового менеджера, оно интегрируется с различными компонентами Android,
создавая ощущение, что это как будь-то часть операционной системы.
Кроме того, оно предоставляет пользователю возможность экспортировать данные из других приложений и сохранять их в виде файлов.

Возможности:

- Создание папок и свободная организация файлов.
- Все файлы в частном хранилище не будут отображаться в других приложениях.
- Доступ через системное приложение «Файлы» (DocumentsProviderUI).
- Дополнительная ярлык быстрого доступа для устройств, на которых не доступно системное приложение «Файлы».
- Блокировка доступа к личному хранилищу.
- Быстрая блокировка.
- Автоблокировка через 15 минут.
- Пароль для блокировки доступа к файлам.
- Импорт данных с помощью функции «Поделиться» в Android
